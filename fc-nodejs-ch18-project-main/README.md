# Fastcampus Node.js Chapter18 Project

- 패스트캠퍼스 온라인 강의
    - 한 번에 끝내는 Node.js 웹 프로그래밍 초격차 패키지 Online 강의의 백엔드 Part2의 Chapter18 프로젝트에 해당하는 Repo 입니다.

- 터미널로 프로젝트 루트로 오신 후 다음을 실행해 주세요.
    ```bash
    npm install
    ```

- `.env` 파일을 만들어 프로젝트에서 사용되는 환경변수들을 설정해 주세요.

    ```bash
    FB_APP_ID=
    FB_CLIENT_SECRET=
    NAVER_CLIENT_ID=
    SERVER_SECRET=
    MONGO_PASSWORD=
    MONGO_CLUSTER=
    MONGO_USER=
    MONGO_DBNAME=
    ```
